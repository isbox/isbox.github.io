值

数组

与其他强类型语言不通，javascript数组可以容纳任何类型的值，如：a = [1, '2', [3], {4: 4}];

对于声明后的数组即可加入值，无需预设大小

数组通过数字进行索引，但是数组也是对象，也可以包含字符串属性

⚠️注意：如果数组的字符串键能被强转为十进制数，那么它会被当作数字索引处理

类数组

一些DOM查询操作会返回DOM元素列表，它们并非真正的数组；另一个是arguments对象（类数组，将函数参数当作列表来访问，es6开始已废止）

工具函数slice可以用于转换这些非数组

字符串

字符串与数组十分相似，它们都是类数组，都有length属性以及indexOf，concat等方法；

在javascript中，字符串是不可变的，而数组是可变。

字符串不可变指字符串的成员函数不会改变其原始值，而是创建并返回一个新的字符串

数组的成员函数则都是在其原始值上进行操作

数字

javascript只有一种数值类型：number，包括“整数”和带小数点的十进制数；

其中，“整数”与其它语言不同，javascript中没有真正意义上的整数，javascript的整数实际上就是没有小数点的十进制数；

特殊的数字

如果数学运算的操作数不是数字类型，会返回NaN；

但是，NaN是不是数字的数字，因为typeof NaN; // number

是唯一一个自反的值NaN === NaN; // false，es6可以使用Object.is来判断相等

可以使用 window.isNaN / Number.isNaN(es6)，来判断是否为NaN；

无穷数

JavaScript 使用有限数字表示法，运算结果有可能溢出，此时结果为 Infinity（ Number.POSITIVE_INfiNITY） 或者 -Infinity（Number.NEGATIVE_ INf iNITY）

在 js 中，0作为被除数也会返回 Infinity

0值

javascript中有一个常规的0值（+0），还有一个 -0，普通加减法不会得到 -0 值

值和引用

在许多编程语言中，赋值和参数传递可以通过值复制、引用复制来完成；

javascript 引用的指向是值。如果一个值有10个引用，这些引用都只会指向同一个值，相互之间没有引用 / 指向关系；

javascript对值和引用的赋值 / 传递完全根据值类型来决定

简单值（总是通过复制来赋值/传递）包括：

null, undefined, 字符串, 数字, 布尔值, symbol

复合值 — 对象（总是通过引用复制的方式来赋值/传递）包括

数组, 封装对象, 函数

上面的情况在函数中经常会让人困惑

基本类型是不可修改的
