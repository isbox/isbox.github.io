类型

javascript内置类型，除了对象，其它统称为“基本类型”

  • 空值（null）
  • 未定义（undefined）
  • 布尔值（boolean）
  • 数字（number）
  • 字符串（string）
  • 对象（object）
  • 符号（symbol）es6新增

null需要使用复合条件来检测，null也是基本类型中唯一一个“假值”（或 false like）

对于typeof function() { ... } === function也为真，这样看来虽然函数也像是js中一个内置类型，而实际上函数是object的“子类型”，具体来说

函数是可调用对象，它有一个[[Call]]内部属性，该属性使其可以被调用

函数中的length属性指的是其声明的参数数量

值和类型

javascript中的变量是没有类型的，只有值才有，变量可以随时持有任何类型的值。

当对变量进行typeof操作得到不是变量的类型，而是值得类型

变量在未持有值时为undefined

多数开发者倾向于将 undefined 等同于 undeclared(未声明)，但在 JavaScript 中它们完全是两回事。

  • 已在作用域中声明但还没有赋值的变量，是 undefined 的
  • 没有在作用域中声明过的变量，是 undeclared 的

可以利用typeof的特殊保护机制做变量声明检查
